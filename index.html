Confidence Interval Simulation

General Explanation

This activity demonstrates how confidence intervals change as a result of changes in sample size and confidence level. The user chooses a sample size between 10-300 by dragging the slider. Then the user will select the confidence level for the confidence interval they want. A higher confidence level indicates a higher probability that the true mean falls within the confidence interval. Finally, the user will click the "Run" button to see the effect of their choices on the confidence interval. As a result, two graphs will appear on the screen. In the upper graph, all distributions whose means could be included within the confidence interval around the sample mean will appear, first those with larger means (to the right of the central distribution) and then those with smaller means (to the left of the central distribution). The two vertical lines that appear indicate the means of the most extreme distributions included within the confidence interval. In each run, the last run will be added to the lower graph with the confidence interval obtained based on the data selected by the user.

Operating Instructions

In the first stage, the user will choose a fixed sample size (for example 30) and see how the confidence level changes the size of the confidence interval. Pay attention - Do the distributions change in their shape? What changes? How does the confidence interval change as a result of changing the confidence level? In the second stage, the user will choose a certain confidence level (for example 95%) and see how changes in sample size affect the confidence interval. What changes now? Experiment with the activity further and try to examine different extreme cases, extreme sample sizes and varying confidence levels. Do you identify certain patterns in these changes
